Mos.Editor.Tab

Description

Handles the tab ui element within the editor as well as a file it represents.

Properties

MosEditor_Tab tab.panel

The vgui tab element used in the editor

string tab.file

The path to the file being handled

Methods

tab:Close()

Closes the tab

tab:Focus()

Focuses the tab in the editor

string tab:SetName()

Sets the name of the tab to show in the editor
