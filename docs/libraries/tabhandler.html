Mos.Editor.TabHandler

Description

The tab handler handles all tabs created and manages user input to make sure a tab is always available.
You can use this library to create, remove and access tabs.

Methods

Tab TabHandler.AddTab( string name, string path )

Creates a new tab, assigning it a name and a file to handle

Tab TabHandler.GetTabByFile( string path )

Attempt to get a tab by file path
