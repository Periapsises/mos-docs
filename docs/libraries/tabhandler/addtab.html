Tab Mos.Editor.TabHandler.AddTab( string name, string path )

Description

Creates a new tab with the specified name and file.
The file is opened and the tab is focused in the Editor automatically.

If a tab with the same file exists, it will be reused and focused.

Arguments

string name

The name that will be shown on the tab

string path

The path to the file that the tab handles

Returns

Tab

The tab that was created or the tab with the same file if one existed
